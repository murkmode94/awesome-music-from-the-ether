# Contributing Guide
Thanks for helping improve this trusted producer resource hub.

## What to Contribute
- Official plugin/vendor pages
- Well-maintained audio development resources
- High-quality educational references for production and DSP
- Corrections to broken or outdated links

## Submission Requirements
Every submission should include:
- Resource name
- Official URL
- Short reason for inclusion
- License/pricing context when relevant (free, paid, open source, etc.)

## Pull Request Rules
- Keep descriptions concise and factual
- Avoid marketing-style language
- Do not submit unverified mirror links
- Keep one logical change per pull request when possible

## Quality Standard
This repo is for trusted, professional use.  
If a source cannot be verified, it should not be merged.
